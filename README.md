# 📊 Global Superstore Sales Analysis Dashboard

## 🌐 Project Overview

This project presents an interactive Tableau dashboard built using the Global Superstore dataset (2011–2014). The dashboard provides comprehensive business insights into sales performance, profitability, customer segments, and product category trends to support data-driven decision-making.

## 🎯 Project Objectives

- Analyze sales and profit performance.
- Identify high-performing product categories.
- Evaluate customer segment contributions.
- Track regional sales trends.
- Visualize key business KPIs.

## 📈 Key Performance Indicators (KPIs)

- Total Sales: ₹12.64M
- Total Profit: 1.46M+
- Average Discount: 14.29%
- Maximum Sales: 22,638

## 📊 Dashboard Components

### Sales Distribution by Segment
- Consumer Segment
- Corporate Segment
- Home Office Segment

### Customer Segment Impact Analysis
- Segment-wise category contribution
- Product category performance analysis

### Revenue Performance Analysis
- Technology category revenue trends
- Sub-category comparison

### Sales Trend Analysis
- Regional sales growth over time
- Running sales performance tracking

### Waterfall Analysis
- Sub-category contribution to total sales
- Running total sales evaluation

## 💡 Business Insights

- Consumer segment generated the highest sales contribution.
- Technology products delivered strong revenue performance.
- Regional sales showed consistent growth across the analysis period.
- Chairs and Storage contributed significantly to overall sales.
